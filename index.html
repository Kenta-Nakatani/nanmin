🌍 世界難民分布マップ

2020-2024年 グローバル難民データの可視化

📊 プロジェクトについて

HDX (Humanitarian Data Exchange) HAPIから取得した2020年から2024年までの世界の難民データを可視化したプロジェクトです。

  • データ期間: 2020年1月1日 - 2024年12月31日
  • データソース: HDX HAPI Refugees Global Dataset
  • データサイズ: 約60万レコード
  • 技術: HTML5, CSS3, JavaScript, Plotly.js
🚀 マップを開く 📁 GitHub

🎯 主な機能

  • インタラクティブ世界地図: 国別難民数の可視化
  • 統計情報: リアルタイム統計データ
  • 移動フロー: 出身国から避難先国への移動ルート
  • 年別推移: 2020-2024年の変化
  • 年齢・性別分析: 詳細な人口統計
